# 住房出租 MVP 规划

## Goal

在共享房产底座和已验证的民宿占用能力上，建立面向集中式公寓和园区人才公寓的整套住房长期出租闭环。

## Dependencies

- 依赖 `07-24-scope-uat-alignment`。
- 依赖 `07-24-shared-property-foundation`。
- 在民宿 MVP 之后实施，并复用已验证的整套房源占用和模式切换能力。

## Users

- 公寓/人才公寓运营人员
- 招租人员
- 合同审批人员
- 现场交割人员
- 财务人员
- 维修/物业人员
- 审计/管理员

首期不提供公开找房、租客 App/小程序或在线签约。

## Core Flow

1. 将整套经营房源切换为长租模式。
2. 建立个人租客和同住人档案。
3. 登记看房、申请或直接创建住房租约草稿。
4. 配置租期、租金周期、押金和费用项目。
5. 审批租约，线下签署并上传签署件。
6. 租约生效并锁定完整租期占用。
7. 完成入住交割、表底读数和物品清单。
8. 生成租金、周期费用和其他应收。
9. 人工登记收款、核销、费用调整和退款。
10. 处理报修、耗材采购、租客收费和成本归集。
11. 续租、变更或退租，完成验收、结算和押金退还。

## MVP Scope

### 经营范围

- 集中式公寓和园区人才公寓。
- 只支持整套出租。
- 分散式房东、产权委托、房东分账和平台佣金暂缓。

### 租客

- 个人租客、紧急联系人和同住人。
- 身份证件、手机号等敏感字段受权限、脱敏和审计保护。
- 个人租客不是系统 SaaS 租户，也不强制复用企业 `biz_park_tenant`。

### 租约

- 草稿、待审批、待签署、有效、即将到期、已到期、退租中、已终止、已作废。
- 支持月、季、半年、年及自定义若干月支付周期。
- 保存首期账单日、每期应收日、租金金额和尾期处理方式。
- 线下签署或外部签署后上传 PDF；暂不接电子签约平台。
- 续租、变更和退租保留历史及审计。

### 入住与退租交割

- 入住交割清单、家具家电、钥匙门卡、照片、表计读数和签字附件。
- 退租验收、损坏赔偿、未结费用、押金抵扣和退款。
- 退租完成后释放长租占用，并按经营决策切换为待整理、长租可用或短租准备状态。

### 费用

- 租金、押金、水费、电费、燃气费、物业费和其他周期费用。
- 水电优先复用能源表计和读数，同时允许人工录入用量、单价或金额。
- 账单保留计费周期、起止读数、用量和调整记录。
- 不接外部水电公司，不做自动代扣。

### 采购与转收费

- 轻量采购单：供应商、日期、项目/房源、费用分类、明细、数量、单价、金额、票据、审批和付款状态。
- 运营采购成本与租客应收分账。
- 采购成本可通过受控操作生成租客代购/赔偿应收，并保留来源。
- 暂不建设仓库、库存、入库、领用、调拨和盘点。

### 财务与服务

- 人工收款登记、核销、调整、冲销、退款和押金处理。
- 复用现有应收、支付、文件、审计、工单和能源底座。
- 租客报修由运营后台/现场端代录或未来租客服务入口接入。

## Acceptance Criteria

- [ ] 可从空闲房源完成个人租客建档、租约、签署登记、生效、入住、账单、收款和退租闭环。
- [ ] 有效租约期间不能创建短租占用或另一有效租约。
- [ ] 周期账单金额、时间和来源可追溯。
- [ ] 能源表计账单与人工录入均可审计。
- [ ] 运营采购成本与租客应收严格分账，转收费保留来源。
- [ ] 押金抵扣、退款、减免和赔偿保持财务审计性。
- [ ] 管理后台和 390px 级现场移动端可完成核心操作。

## Out of Scope

- 分散式房东、委托运营、房东结算、平台佣金。
- 房间和床位出租。
- 公开找房、租客 App/小程序和在线申请。
- 在线支付、自动代扣和电子签约。
- 复杂阶梯租金、营业额提成和自动涨价。
- 完整采购库存系统。
