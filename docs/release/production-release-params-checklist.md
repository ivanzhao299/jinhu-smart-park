# JinHu Smart Park 生产发布参数确认清单

## 1. 目的

本清单用于生产发布前确认所有必要参数，避免发布当天临时补信息，减少因参数缺失或误填导致的发布风险。

## 2. 使用原则

- 本文档不记录真实密钥。
- 敏感参数只记录确认状态、保管人和存放位置。
- 所有真实值应保存在受控密钥系统、运维密码库或受控环境变量文件中。
- 发布前必须由发布负责人和运维负责人共同确认。

## 5. 认证与安全参数确认

- 账号密码登录为首发唯一登录方式。
- 短信验证码登录禁用。
- 微信扫码登录禁用。
- `AUTH_SMS_FIXED_CODE` 必须为空。
- `AUTH_SMS_CODE_VISIBLE=false`。
- `AUTH_WECHAT_MOCK_ENABLED=false`。
- `JWT_SECRET` 已由运维生成并保管。
- 首个管理员密码不写入文档。
