# 统一认证中心后续记录

## 当前已落地

- 账号密码登录。
- 手机验证码登录基础闭环。
- 微信 OAuth 登录骨架：支持授权 state、回调、未绑定账号提示、绑定接口预留。
- Refresh Token、退出登录撤销、登录审计。
- 统一身份表 `sys_user_identity`，可承载 password、mobile、wechat_open 等身份来源。

## 后续必须补齐

- 接入真实短信服务商，替换开发环境 mock 验证码。
- 增加账户安全页面：绑定 / 解绑手机号、微信、后续企业微信 / 钉钉 / 飞书身份。
- 增加登录策略配置：是否允许密码、手机、微信、第三方 SSO 登录。
- 增强风控：验证码发送频控、错误次数锁定、IP 限流、异常登录审计。
- 扩展企业微信、钉钉、飞书等 provider，继续复用 `sys_user_identity`。
