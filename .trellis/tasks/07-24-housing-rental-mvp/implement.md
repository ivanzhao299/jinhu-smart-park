# 住房出租 MVP 实施计划

## 1. 批次

### A. 租客与租约

- [ ] 个人租客、联系人和同住人。
- [ ] 住宅租约草稿、审批、签署登记和生效。
- [ ] 共享占用接入。

### B. 交割与押金

- [ ] 入住交割、物品、照片、表底和凭证。
- [ ] 押金台账。
- [ ] 退租验收、抵扣和退款。

### C. 周期费用

- [ ] 租金周期计划和应收生成。
- [ ] 水电燃气物业费用。
- [ ] 能源表计和人工录入双路径。

### D. 采购成本

- [ ] 采购单、明细、票据和审批/付款状态。
- [ ] 项目/房源成本归集。
- [ ] 受控转收费和来源追溯。

### E. 服务与 UAT

- [ ] 报修/工单联动。
- [ ] 后台与 390px 移动端闭环。
- [ ] 从租客建档到退租结算的 E2E。

## 2. 回滚点

- 住宅租赁使用独立模块和表，不修改现有商业合同语义。
- 新费用来源通过适配层进入财务底座。
- 模块关闭后不影响商业租赁和民宿。
- 财务回滚使用反向记录，不删除历史。

## 3. 必测场景

- 同房源租期冲突和与短租占用冲突。
- 不同支付周期、首尾期和跨月边界。
- 表计账单与人工账单。
- 押金抵扣、赔偿、退款和减免。
- 采购成本与租客应收分账。
- 租户/项目隔离和个人信息权限。
